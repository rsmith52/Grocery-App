Basic things about Angular

  • This file is found under app and is the html template of app.component

  • Components are like objects, but we can have separate object types too that aren't components

  • This app is currently set up to be run with node.js and express as the server, but can be switched

  • Run the app on localhost:4200 with 'ng serve' or build the app with 'ng build', then run the app through the Express server on localhost:1337 with 'nodemon server.js'

  • 'ng serve' is better for testing, but in actual hosting it will be run off of the express server, or whatever we use
